# Gestión Médica

Aplicación de gestión de pacientes, médicos y citas médicas.

## Tecnologías
- **Frontend:** React + Tailwind CSS
- **Backend:** Node.js + Express + SQL Server
- **Base de Datos:** SQL Server 2019

## Instalación

### Backend
```bash
cd backend
npm install
npm start
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Características
- CRUD de pacientes
- Gestión de médicos
- Agenda de citas
